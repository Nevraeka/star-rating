Events

ratingUpdated

rating updated is a DOM Custom Event that fires whenever the value of the <star-rating> is updated. This event passes a JavaScript object with 2 key/value pairs to 'event.detail'.

Event Detail data

maxValue - this is the current maxvalue attribute value at the time the event fires

value - this is the current value property/attribute value at the time the event fires

Attributes

value

value is mapped directly to the value property & is meant to be a 'output only' attribute. This means it is internally managed by the element and should be considered 'read-only'

Properties

value

value represents the number of stars currently selected. Value directly maps to the value attribute (see above).

Methods

reset

this method resets the value of the <star-rating> element to 0 and triggers the ratingUpdated event.
